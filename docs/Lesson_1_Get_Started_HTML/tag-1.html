Cheat Sheet (HTML Basics)

<p></p> ⇒ Paragraph tag 

<h1></h1> ⇒ Heading 1 tag

<h2></h2> ⇒ Heading 2 tag

<h3></h3> ⇒ Heading 3 tag

<h4></h4> = Heading 4 tag

<h5></h5> ⇒ Heading 5 tag

<h6></h6> ⇒ Heading 6 tag

<br/> ⇒ Line break 

<strong></strong> ⇒ Bold text 

<i></i> ⇒ Italic text

<a></a> ⇒ Link tag 

- href ⇒ link destination

<img/> => image tag 

- src ⇒ source of image
- height ⇒ height of image, pixels, em etc
- width ⇒ width of image, pixels, em etc

<ul></ul> ⇒ Unordered list

<ol></ol> ⇒ Ordered list 

<li></li> ⇒ List item   

<!DOCTYPE html> ⇒ First line in all HTML files

<html></html> ⇒ Wraps all HTML code on HTML file

<head></head> ⇒ Holds css link files, title, fonts etc. Hidden

<title></title> ⇒ Title of page, shows in the browser tab

<body></body> ⇒ Holds all visible elements in HTML file
