    This is a heading 1 Tag

    This is a heading 2 Tag

    This is a heading 3 Tag

    This is a heading 4 Tag

    Link

    <a>This is a link tag</a>

    This is a link tag, hover over me to see

    Link with Attributes

    <a href="https://www.google.com/" target="_blank">
This link tag will open a page
</a>

    This link tag will open a page.
    It has a href="" for a link desitination
    It has target="_blank" to open new tab

    Image

    <img src="https://bit.ly/2TdOyRs" width="200px" height="200px"/>

    This is an image
    It has a src attribute which is a path to the exact file, in this case the image
    It has set attributes of width & height in pixels

    Unordered List

    <ul>
    <li>A list item, of an unordered list</li>
    <li>Another one</li>
    <li>A third one</li>
</ul>
    • A list item, of an unordered list
    • Another one
    • A third one

    This is an unordered List
    With <li> as list items
    Watch the order here, it has do with HTML parent/child structure
    We will be learning that next

    Ordered List

    <ol>
    <li>A list item, of an ordered list</li>
    <li>Another one</li>
    <li>A third one</li>
</ol>
    1. A list item, of an ordered list
    2. Another one
    3. A third one
    This is an ordered List
